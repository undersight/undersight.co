## Running locally

### Setup

1. `npm install --global gatsby-cli`
2. `yarn`

### Develop

1. Run server
  - Live development:

    `gatsby develop`

  - Live development with an access on local network:

    `gatsby develop -H your-local-ip-address`

  - Browse the build:

    `gatsby build && gatsby serve`

2. http://localhost:8000
